#  ORBITEYE
## Central Espacial de Monitoramento Climático

---

#  Descrição do Projeto

O OrbitEye é um sistema desenvolvido em Python com interface gráfica que simula uma central espacial de monitoramento climático em tempo real.

A aplicação utiliza uma API climática para coletar dados meteorológicos de cidades e gerar alertas automáticos relacionados a possíveis riscos ambientais, como incêndios, enchentes, tempestades e frio extremo.

O projeto foi inspirado nos modernos sistemas de monitoramento utilizados na indústria espacial e em tecnologias de sensoriamento remoto via satélite.

---

#  Objetivo da Solução

O principal objetivo do OrbitEye é demonstrar como sistemas computacionais podem utilizar dados climáticos globais para auxiliar no monitoramento ambiental e na geração de alertas automáticos.

O projeto busca representar, de forma simplificada, como tecnologias espaciais e sistemas de monitoramento remoto podem contribuir para:

- Monitoramento climático
- Prevenção de desastres ambientais
- Análise de dados em tempo real
- Tomada de decisão baseada em informações meteorológicas

#  Instalação do Projeto


```bash

```

---


---

#  Como Executar

Execute o arquivo principal:

```bash

```

#  Exemplos de Uso

## Exemplo 1
Digite uma cidade:

```txt
São Paulo
```

- Temperatura
- Umidade
- Vento
- Condição climática
- Alertas automáticos
